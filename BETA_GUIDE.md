## 2. 어떻게 쓰는 앱인가요?

세 가지 핵심 컨셉:

### ① 나만의 색깔 표현
- AI 인터뷰로 3분이면 프로필 완성
- 8가지 컬러로 성격 표현 (오렌지/블루/핑크/그린/퍼플/옐로/레드/그레이)

### ② 지인 네트워크 매칭
- 다른 데이팅 앱과 다른 점: **모르는 사람보다 친구의 친구**부터
- 1촌 (직접 친구) → 무료
- 2촌 (친구의 친구) → 3,000원
- 더 멀어질수록 비싸짐 → 신중한 매칭 유도

### ③ 주선 시스템
- 친구를 주선해주면 매칭 성공 시 **포인트 보상** (30~50%)
- 레벨업할수록 커미션 비율 증가

---

## 3. 가장 먼저 해볼 것

### Step 1. 프로필 만들기 (5분)
1. 베타 코드 입력
2. 카카오 로그인
3. 기본 정보 → 사진 업로드 → AI 인터뷰 → 이상형
4. AI가 프로필 글 자동 생성 → 확인 후 완료

### Step 2. 메인 피드 둘러보기
- "AI 시그널" — 매일 1명 무료 추천
- 시드 유저 12명 (베타 테스트용 가짜 프로필) 미리 채워둔 상태
- 마음에 드는 사람 카드 탭 → 프로필 보기

### Step 3. 주선자 마켓플레이스 (선택)
- 활동 중인 주선자 5명 (베타 시드)
- 본인이 주선자 되고 싶으면 마이페이지 → 주선자 활성화

### 작동하지 않거나 mock 인 기능

- 사진은 매칭된 분께만 공개되도록 추후 강화 예정입니다 (현재는 URL 직접 노출 가능)
- 실제 매칭/결제 흐름은 시뮬레이션 위주입니다
- 데이터는 정식 출시 시 그대로 이관됩니다 (베타 동안 가입한 계정 유지)

4. **주선 시스템** — 친구를 주선하는 컨셉이 매력적인가요? 부담스러운가요?

## 7. 자주 묻는 질문

### Q. 본인 사진 올리기 부담스러워요
A. 베타 동안은 본인 사진이 권장이지만 필수는 아니에요. 풍경/일러스트 같은 걸로 시작해서 신뢰가 쌓이면 변경 가능합니다.

### Q. 베타가 언제 끝나나요?
A. 약 2-4주 진행 예정. 정식 출시 직전에 안내드릴게요.

### Q. 친구도 초대할 수 있나요?
A. 베타 코드 공유해주시면 OK. 단, 너무 많은 분이 한번에 들어오시면 시스템 부하 우려가 있으니 한 명씩 알려주세요.

### Q. 데이터 삭제하고 싶어요
A. 마이페이지 → 회원탈퇴. 또는 직접 알려주시면 즉시 처리해드립니다.

### Q. 카카오 로그인이 안 돼요
A. 베타 카카오 앱 설정 문제일 수 있어요. 우선 이메일 가입으로 진행 부탁드리고, 알려주시면 즉시 수정합니다.
