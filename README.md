# Palette 🎨

> "나의 색을 찾고, 너의 색과 조화를 이루다"
> "Find Your Color, Meet Your Match"

**Palette**는 지인 네트워크 기반의 신뢰할 수 있는 주선 데이팅 서비스입니다.
화가가 팔레트에서 자신의 그림에 어울리는 색을 고르듯, 우리 모두는 각자의 개성, 취향, 이상형이라는 고유한 색을 가지고 있습니다.

## 🎯 핵심 가치

### 1. 나만의 색깔 표현 (Express Your Color)
- **AI 인터뷰**로 나의 개성, 취향, 이상형을 명확히 정의
- **동영상/사진 신뢰도 시스템**으로 진짜 나를 보여주기
- **외부 공유 가능한 개인 프로필 페이지**
- *"내가 어떤 사람인지, 어떤 사람을 원하는지 명확해진다."*

### 2. 조화로운 매칭 (Find Your Harmony)
- **1-2촌 지인 네트워크**에서 나와 어울리는 색 찾기
- **주선자 승인 → 수신자 승인** 2단계 검증
- **제한된 선택지 + 건별 과금**으로 신중한 선택
- *"많은 매칭이 아닌, 나와 맞는 한 사람"*

### 3. 의미있는 관계 (Create Masterpiece)
- **사회적 책임감**으로 진지하게 대함
- **쉽게 포기하지 않는 구조**
- **주선자 인센티브**로 생태계 활성화
- *"적은 매칭이지만 깊고 의미있는 관계"*

## 🚀 주요 기능

### Phase 1: AI 활용 나만의 색깔 표현 (0-3개월)

#### 1. AI 인터뷰 기반 프로필 생성
- ChatGPT 스타일의 대화형 인터페이스
- 10-15개 질문으로 나의 색깔(프로필) 자동 완성
- 사용자 답변 기반 자동 텍스트 생성
- 8가지 색깔 메타포로 개성 표현 (오렌지, 블루, 레드, 핑크, 그린, 퍼플, 옐로우, 그레이)

#### 2. 사진/동영상 신뢰도 시스템
- **Bronze/Silver/Gold** 등급 (0-100점)
- AI 이미지 분석: 타인 촬영, 전신 사진, 얼굕 명확도
- 동영상 업로드 시 +50점 (고신뢰도)
- 과도한 보정 감지 및 감점

#### 3. 색깔 공유 링크
- 외부 공유 가능한 개인 프로필 페이지
- QR 코드 생성
- SNS 최적화 미리보기
- 조회 통계 및 분석

### Phase 2: 조화로운 만남 주선 (3-6개월)

#### 1. 지인 네트워크 연결
- **1촌**: 직접 친구 (전화번호 연동, 초대 코드)
- **2촌**: 친구의 친구
- **3촌**: 더 먼 관계
- 프라이버시 설정으로 노출 범위 제어

#### 2. 프로필 탐색 및 열람
- **1촌**: 무료 열람
- **2촌**: 3,000원 (주선자 30% 수익)
- **3촌**: 5,000원 (중간 주선자들 배분)

#### 3. 2단계 주선 프로세스
```
요청자 → [주선 요청] → 주선자 승인/거절
                          ↓
                    수신자 승인/거절
                          ↓
                    매칭 성공 → 연락처 교환
```

#### 4. 리워드 시스템
- **레벨별 커미션율**: Lv.1 (30%) → Lv.5 (50%)
- 프로필 열람 시 1차 수익
- 매칭 성공 시 2차 수익 (주선 요청권 3만원)
- 포인트 → 현금 출금 가능 (10,000P 이상)

#### 5. 관계 발전 촉진
- 매칭 후 10일 쿨타임 (집중 유도)
- 주선자 응원 메시지
- 관계 단계 트래킹 (선택)

### Phase 3: AI 매칭 확장 (6-12개월)

#### 1. AI 추천 알고리즘
- Collaborative Filtering + Content-Based Filtering
- 매칭 점수 및 추천 이유 제공
- 행동 기반 학습

#### 2. 그룹 매칭
- 3명씩 그룹 매칭 (3:3)
- 오프라인 만남 주선
- 장소/시간 자동 추천

## 🏗️ 기술 스택

- **Language**: Kotlin 2.2.21 with Java 21
- **Framework**: Spring Boot 4.0.1
- **Database**: H2 (dev/test), MySQL (production)
- **Security**: Spring Security with OAuth2
- **Architecture**: Domain-Driven Design (DDD)
  - Domain Layer: 순수 비즈니스 로직
  - Persistence Layer: JPA 엔티티 및 레포지토리
  - Application Layer: 서비스 및 유스케이스
  - Presentation Layer: REST API

## 📦 프로젝트 구조

```
palette/
```

## 📋 주요 도메인 모델

### User (사용자)
- **REGULAR**: 일반 회원 (매칭 받을 수 있고, 주선도 가능)
- **MATCHMAKER_ONLY**: 주선 전용 (자신은 매칭 안 받음)
- OAuth 인증, 프로필 완성 여부, 약관 동의

### Profile (프로필)
- 기본 정보 (키, 체형)
- 직업, 학력, 거주지
- 자기소개, 관심사
- 라이프스타일 (흡연, 음주, 종교)
- 이상형 설정
- Color Type (8가지 색깔 메타포)
- 신뢰도 점수 (0-100)

### Matchmaker (주선자)
- 레벨 시스템 (1-5)
- 커미션율 (30-50%)
- 통계 (총 요청, 승인, 거절, 성공, 실패)
- 포인트 (총 포인트, 출금 포인트, 보류 포인트)

### ProfilePhoto (프로필 사진)
- AI 신뢰도 분석
- Trust Factor (셀카/타인 촬영/불명확)
- 품질 점수, 얼굴 인식, 전신 여부
