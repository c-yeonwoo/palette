# 0027 — 소개(홈) 피드 내부 탭 + 카드 통일 + 색 커버 복귀 + 타입 스케일 압축

- **상태**: Accepted
- **결정일**: 2026-06-07
- **결정자**: ys.choi

## 컨텍스트

소개(홈) 화면 피드백:
- 오늘의 추천(가로 스크롤 w-40)과 지인 카드(2-col grid)의 크기가 달라 들쭉날쭉.
- AI 섹션이 `px-4` 컨테이너 안에서 또 `px-4`를 줘 좌우 inset이 32px로 어긋남(컬러 궁합 배너는 `mx-4`라 또 달랐음).
- 작은 글씨(9~11px)가 너무 안 보임. 최대·최소 폰트 격차가 큼.
- 카드 커버가 블랙+골드라 경쟁 앱(골드스푼) 느낌.
- "~ 외 N명 통해" 문구가 어색.

## 결정

1. **내부 탭 분리** — 피드를 `오늘의 추천 | 지인` 세그먼트 탭으로 분리. 필터 적용 시 자동으로 `지인` 탭 전환.
2. **카드 크기 통일** — 두 탭 모두 `grid grid-cols-2 gap-3` 동일 그리드(aspect-3/4). 가로 스크롤 폐기. 페이지 좌우 padding은 16px(`px-4`) 단일 적용.
3. **컬러 궁합 배너 inset 정렬** — `DailyMatchBanner`의 `mx-4` 제거하고 상위 `px-4`가 inset 통제(카드와 동일 16px). 위 여백 `pt-6`로 키워 위치 내림.
4. **카드 커버 = 색 물감 워시 (블랙골드 폐기)** — 잠긴/미오픈 커버 배경을 상대의 컬러타입 hex 기반 라이트 워시로 교체(`radial-gradient(color) + linear-gradient(surface-sunken→muted)`). 중앙에 색 원 + "탭해서 열기"(foreground/55). AI 뱃지·CTA는 `bg-brand-soft text-gold-strong`로. (직전 버전의 `paint-overlay.png`는 리포에 커밋된 적이 없어 자산 복원 불가 → 색 기반 커버로 재구성 = 팔레트 컨셉에 부합.)
5. **타입 스케일 압축** (design system, `theme.css`):
   - 최소 상향: 하드코딩 `text-[9/10/11px]` → `text-xs`; `--text-xs` 12→13px; `.text-caption` 12→13px.
   - 최대 하향: `--text-2xl` 24→22px, `--text-3xl` 30→26px.
   - 범위 13~26px로 좁힘.
6. **문구** — "{이름} 외 N명 통해" → "{이름}님의 친구" / "{이름}님 외 N명의 친구".

## 결과

- 오늘의 추천·지인 카드 동일 크기, 좌우 16px 정렬.
- 카드가 사용자의 '색'을 노출 → 팔레트 정체성 강화, 골드스푼 느낌 제거.
- 최소 폰트 13px floor로 가독성 개선.
- vite build / vitest(40) 통과.

## 후속

- 타입 스케일 변경은 전 화면 영향 → 타이트한 칩/배지에서 오버플로 있으면 개별 조정.
- `paint-overlay.png` 텍스처가 필요하면 자산 추가 후 커버에 옵션 합성 가능.
