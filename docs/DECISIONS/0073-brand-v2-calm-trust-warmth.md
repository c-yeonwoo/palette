# 0073 — 브랜드 v2: "차분한 신뢰 + 따뜻한 온기" (인격 단일화)

- **상태**: Accepted
- **작성일**: 2026-07-13
- **작성자**: ys.choi
- **선행**: ADR 0001(모노 차콜 무대), 0067(골드→코랄), 디자인 토큰 P1~P10(theme.css)
- **동기**: 브랜드 인상이 "프리미엄 vs 귀여움 vs young" 으로 섞여 일관되지 않다는 대표 진단 → 인격을 하나로 좁힌다.

---

## 1. 문제 — 세 층이 다른 브랜드를 가리킨다

디자인 토큰 시스템 자체는 잘 구축됨(3-레이어). 문제는 **방향**이다. 세 층이 각자 다른 인격을 지시:

| 층 | 지시하는 인격 | 근거(코드 실측) |
|---|---|---|
| ADR 0001 (기반) | 프리미엄 절제 | 모노 차콜, radius 토큰 4–12px("둥글=캐주얼 지양"), 절제 음영 |
| ADR 0067 (전환) | 따뜻·친근 | brand 코랄 #E27A57, §4 "프리미엄 인상 약해짐 / 모서리·타이포 범위 밖(미완)" |
| 실행 드리프트 | young·cute | emoji 30개 파일, `rounded-full`×276 + `rounded-2xl`×134, 마스코트 '팔리', 8색 채도 92% |

토큰은 "각지게=프리미엄"이라 선언했는데 화면엔 둥근 모서리 410곳. 브랜드는 "탈스펙 따뜻함"인데 표면엔 골드 프리미엄 잔재 + 이모지 young. → "고급인데 귀엽고 20대 앱 같은" 혼선.

## 2. 결정 — 북극성 하나

> **"차분한 신뢰 + 따뜻한 온기" — 어른스러운 미니멀에 사람 냄새 한 스푼.**
> young(귀여움)도, 차가운 럭셔리(골드)도 아니다. 타깃(25–39 직장인)·차별점(신뢰)에 정렬.

포지셔닝: (절제↔표현) × (프리미엄·차가움↔캐주얼·young) 2축에서 **오른쪽 위 = 따뜻하지만 어른스러운 미니멀** 한 점으로 색·모서리·타이포·카피를 모두 수렴.

### 2-1. 브랜드 컬러 — 무르익은 코랄 (방향 A 채택) ⚠️ 2-1b 로 대체됨
0067의 코랄 자산을 지키되 **한 톤 깊고 차분하게**: 살몬빛(young) → 테라코타빛(어른).
- `--brand: 15 72% 60%`(#E27A57) → **`15 54% 49%`(#C25D3A)**. 같은 hue(15) 유지, 채도 72→54·명도 60→49.
- `--brand-soft`/`--brand-strong` 정합 유지(soft 위 텍스트 대비 AA).
- **다크모드**: 어두운 배경 가독성 위해 `--brand` 밝은 코랄 override 신설(라이트/다크 공유 폐기).

### 2-1b. 브랜드 컬러 갱신 — 밝은 골드 (2026-07-13, **최종 lock**)
무르익은 코랄(#C25D3A)이 실사용 리뷰에서 "칙칙하다"는 피드백 → **밝고 상큼한 골드로 전환**.
- 라이트 `--brand: 40 96% 52%`(**#FAA80A**). hue 15(테라코타) → 40(골드), 채도·명도 상향.
- 폰트색 동기화: `--brand-strong 34 90% 36%`(soft/흰 배경 텍스트 AA 5:1+), `--brand-foreground 32 45% 12%`(골드 solid 위 딥브라운).
- 다크 `--brand: 42 92% 62%` / `--brand-strong: 44 92% 70%`.
- **오실레이션 종료 선언**: 0067(골드→코랄) → 2-1(코랄 심화) → 2-1b(코랄→골드). 브랜드 색은 하루에 세 번 바뀜(코랄→오렌지 hue33→골드 hue40). **본 값(#FAA80A, hue 40)을 고정값으로 lock** — 이후 변경은 새 ADR 필요. 정체성 문구('따뜻한 온기'')는 유지, 골드는 온기+생기를 더 밝게 표현.

### 2-2. 8색 채도 진정 (유저의 색은 유지하되 표면은 차분히)
'팔레트=색' 컨셉상 8색(`--ct-*`)은 유지. 단 **가장 시끄러운 -500 채도를 한 단계 낮춰** 무지개가 표면을 뒤덮는 인상 완화(orange/yellow 92%→80% 등). "색은 의미 있는 순간에만 팝" 원칙은 컴포넌트 단계(2차)에서.

### 2-3. young 드리프트 제거 4종 (색과 별개, 어느 방향이든 동일)
1. **이모지 → 아이콘 (선별)**: **UI 크롬·구조 장식** 이모지(🌟🎨👤👥🌱🫂✨ 등, 웰컴/CTA/빈 상태/피처 리스트)만 lucide로 통일. 색타입 이모지 → 컬러 도트/디스크. **메달·등급**(🥇🥈🥉👑💎🏆) → lucide Medal/Crown/Gem/Trophy(색 유지). raw 파스텔 그라디언트(`from-orange-50 to-pink-50` 등) → `bg-brand-soft` 토큰, `animate-bounce` 제거.
   - **유지(북극성 '따뜻한 온기'와 정렬)**: ① **얼굴상 칩 메타포**(🐻곰상·🐶강아지상·🌊데이트스타일·💍목적 등)=의도된 콘텐츠 택소노미(적합 아이콘 부재, 친근함이 차별점). ② **감정 마이크로모먼트**(매칭 성사 🎉·후기 반응 😊🙂😕·위로 🌿). ③ **팔리 챗 보이스**(인터뷰/리포트 페르소나). ④ 별자리·운세 글리프=기능 콘텐츠.
   - **스코프 제외**: `app-admin/*`(내부 운영 도구, 유저 브랜드 무관).
2. **모서리 규율**: `rounded-full`(아바타·칩만) 외 카드·시트는 토큰 8–12px. 토큰 의도 실제 강제.
3. **8색 채도 진정** (2-2).
4. **이중 액센트 정리**: 브랜드는 코랄 하나. 골드는 "프리미엄 시그니처" 지위 **강등** → '골드 큐피드 등급' 이름값으로만 잔존(신규 골드 사용 금지).

### 2-4. 마스코트 '팔리' — 유지 (재검토 결과)
'팔리'는 장식이 아니라 **AI 인터뷰어·색 리포트의 화자 페르소나**로, 북극성의 '따뜻한 온기'를 직접 구현. 격하 대신 **유지** 결정. (챗 보이스의 🎨 등 페르소나 이모지 포함.)

## 3. 롤아웃 (단계적)
1. **1차(본 PR)**: 토큰 — 무르익은 코랄 + 다크 brand + 8색 채도 진정 + 골드 강등(주석). 컴포넌트 로직 변경 0(토큰 참조 312곳 자동 전환). 브라우저 검증.
2. **2차**: 모서리 규율 스윕(`rounded-3xl` 8곳 → 온-시스템 12px).
3. **3차**: 크롬 이모지→아이콘 선별 스윕 + 메달 아이콘화 + 색타입 컬러 디스크 + 파스텔 그라디언트 토큰화.
   - 완료: 홈(FirstTimeGuide·CTA·빈 상태·패스 패널), AiHub(플레이스홀더·가이드), FortuneBanner, LeagueScreen 메달, `matchmakerLevels`(SoT)·주선자 3화면, ColorDetail(컬러 디스크), BetaWelcomeIntro(탈파스텔·탈바운스), ConnectorDashboard 빈 상태.
   - 마스코트: 격하 → **유지**로 정정(2-4).
   - 잔여(후속·저빈도): IntroHistory `💌` 헤더, lib 카탈로그(jobCategory `💻`·feedback `😊`), MyProfile 애착유형 이모지 등. 감정·콘텐츠 이모지는 2-3 유지 원칙에 따라 보존.

## 4. 트레이드오프
- 코랄이 깊어져 채도 화사함↓ — 의도(어른스러움). 되돌리기는 토큰 1값.
- 8색 채도 진정은 시그니처 팔레트 인상을 바꿈 — 1차에서 브라우저 확인 후 조정 여지.
- 다크 brand override 신설로 라이트/다크 코랄이 서로 다른 값(각 배경 대비 최적).

## 5. 영향 범위 (1차)
- `frontend/src/styles/theme.css`(토큰만). 백엔드/DB 무관.
- 검증: `npm run build:mobile` + vitest + 브라우저(홈·프로필, 라이트/다크) computed token 확인.
