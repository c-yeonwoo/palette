# 0067 — 브랜드 컬러: 골드 → 따뜻한 코랄 (정체성 정합)

- **상태**: Accepted
- **작성일**: 2026-06-20
- **작성자**: ys.choi
- **선행**: 디자인 토큰 P1~P10(theme.css), ADR 0064(무현금 주선), 온보딩 콘텐츠 리프레시(#92 탈스펙)

---

## 1. 문제

브랜드 정체성이 "따뜻한 지인 소개 + 탈스펙 + 너의 색(감성)"으로 이동했는데, **디자인 외피는 여전히 앤티크 골드(`#BD9244`) = 럭셔리/노블레스 매칭 시그널**이었다.

- 온보딩에서 외모·학력·경제력(스펙)을 들어내고 "라이프스타일·가치 핏"으로 옮겼는데(#92), 정작 브랜드 색이 결혼정보회사 톤의 골드 → **콘텐츠와 외피의 정체성 불일치.**
- 무현금 전환(ADR 0064)으로 "프리미엄 유료 매칭" 명분도 약해짐.

## 2. 결정

브랜드 메인 액센트를 **골드 → 따뜻한 코랄/테라코타(`#E27450`)** 로 전환. 골드는 **보조 시그니처**(골드 큐피드 등급·소수 프리미엄 포인트)로 강등.

### 토큰 (theme.css)
- `--brand: 15 72% 60%` (#E27450), `--brand-soft: 15 72% 94%` (#FBEAE5), `--brand-strong: 15 56% 37%` (#93442A, 신설 — soft 위 텍스트), `--brand-foreground`: charcoal 유지(코랄 위 대비 ~6:1).
- `--brand` 을 `var(--gold)` 에서 **독립 코랄 값으로 분리**.
- `--gold*` 토큰은 그대로 보존 → 보조 골드 용도.

### 컴포넌트
- `text-gold-strong`(68곳, "soft 위 텍스트" 역할) → `text-brand-strong` 기계 치환. brand-soft(코랄) 위 텍스트가 코랄-strong 으로 정합.
- `PaletteInsightPanel` 의 `gold-strong` 액센트 폴백 → `brand-strong`.
- 로그인·베타게이트 히어로 raw 골드 hex `#C9A86A` → 코랄 `#E8896A`(다크 배경용 밝은 코랄). **첫인상 일관성.**

### 골드로 남기는 것 (의도)
- `MatchmakerRewardScreen` "골드 큐피드" 등급 뱃지(`bg-gold`), `matchmakerLevels.ts` 골드 등급색(`#BD9244`). 등급명이 "골드"라 의미상 골드 유지.

## 3. 근거

- **코랄 = 감성·따뜻·친근·접근성** → "지인이 따뜻하게 이어주는" 정체성과 직결. 골드의 럭셔리/배타 함의 탈피.
- **토큰 아키텍처 활용**: `--brand` 계열 312곳 + `text-brand-strong` 68곳이 한 토큰을 참조 → 값 교체로 전 화면 일관 전환, 컴포넌트 로직 변경 0.
- **접근성**: 코랄 #E27450 + charcoal 텍스트 ~6:1, brand-strong #93442A + brand-soft #FBEAE5 고대비 — AA 통과.
- **'팔레트=색' 컨셉과 비충돌**: 8색 성격 팔레트(`--ct-*`)는 그대로. 코랄은 "브랜드 액센트", 8색은 "유저의 색" — 역할 분리 유지.

## 4. 트레이드오프 / 한계

- **브랜드 인상 변화**: 골드의 "프리미엄/신뢰" 인상이 약해짐. 단 신뢰 축은 지인 네트워크·검증 시스템이 담당하므로 색에 의존하지 않음.
- **다크모드**: `--brand`/`--gold` 는 라이트·다크 공유(기존 골드도 동일). 코랄 mid-tone 은 양쪽에서 액센트로 무난.
- **추가 여지(미채택)**: 모서리(radius 4~8px 샤프)·타이포는 이번 범위 밖. 더 부드러운 톤으로 가려면 별도 검토.

## 5. 영향 범위

- 백엔드/DB 무관. 검증: build + vitest 그린, 베타게이트 스크린샷(코랄) + computed token(`--brand`=#E27450) 확인.
