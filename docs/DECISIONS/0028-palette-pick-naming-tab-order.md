# 0028 — 추천 탭 네이밍 "팔레트 Pick" + 탭 순서(지인 좌 / 팔레트 Pick 우)

- **상태**: Accepted
- **결정일**: 2026-06-07
- **결정자**: ys.choi

## 컨텍스트

소개(홈) 내부 탭에서:
- "오늘의 추천" / "AI 추천"이라는 이름이 진부하고 타 데이팅 앱을 따라한 느낌.
- 팔레트는 지인 네트워크가 핵심인데 추천 탭이 좌측(기본)에 있었음.

## 결정

1. **네이밍: "팔레트 Pick" (Palette Pick).** "오늘의/AI 추천" 폐기. 브랜드명 + 큐레이션 뉘앙스, 색 궁합 컨셉과 결.
   - 탭 라벨, 빈 상태("팔레트 Pick을 준비 중이에요"), 구독 상품명("팔레트 Pick 패스"), 토스트 모두 정렬.
   - **백엔드 식별자는 변경 없음** — 엔드포인트 `/api/v1/feed/ai-signal`, 엔티티 `AiPassSubscription` 등 내부명 유지(표면 네이밍만 교체).
2. **탭 순서: `[지인] [팔레트 Pick]`, 기본 탭 = 지인.** 지인 네트워크가 서비스 핵심이므로 좌측·기본. 팔레트 Pick은 우측 프리미엄 탭.

## 결과

- 홈 진입 시 지인 피드가 먼저, 팔레트 Pick은 우측 탭.
- UI에서 "AI 추천/오늘의 추천" 문구 제거.
- GLOSSARY/POLICY 네이밍 정렬. vite build / vitest(40) 통과.

## 후속

- 백엔드 식별자(ai-signal)를 표면명과 맞추는 리네이밍은 비용 대비 실익 적어 보류(주석/문서로 매핑 유지).
