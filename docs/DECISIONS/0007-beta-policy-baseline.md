# 0007 — 베타 단계 운영 정책 베이스라인

- **상태**: Accepted (베타), 정식 출시 전 재검토
- **결정일**: 2026-05-31
- **결정자**: ys.choi

## 컨텍스트

베타 운영 중 결제 PG 연동, 환불 처리, 광고 도입, 결혼정보업 등록 등 운영 정책 결정 필요.
사용자에게 신뢰 신호를 주는 데이팅 서비스 특성상 정책 명문화가 다른 데이팅 앱보다 더 중요.

## 결정 사항

### 1. 결제 (베타 단계)
- **Toss test 키 그대로 운영** — 실 결제 발생 안 함 (정식 전환 시 live)
- AI Signal 2번째 카드 unlock 은 `paymentKey` 존재만 체크 (Toss 서버 검증 미연동) — Phase 2 에 검증 추가
- 베타 동안 2촌 카드 열람은 무료 (결제 미연동), 정식 전환 시 3,000원 활성

### 2. 환불
- **5분 룰** — 카드 열람 후 5분 내 환불 가능 (오결제 보호), 이후 환불 불가 (정보 소비 완료)
- 시스템 오류로 인한 열람 실패는 즉시 환불 + 1회 무료 보상
- 환불 채널: 베타는 `admin@palette.kr`, 정식은 앱 내 CS

### 3. 광고
- **베타 단계 광고 없음** — 사용자 신뢰 우선
- 마케팅 푸시 없음, `agreedMarketing` 동의 받은 사용자도 시스템 알림만
- 정식 출시 후 B2B 광고 (결혼식장/데이트 코스/선물) vs 절대 없음 vs 프리미엄 구독 검토 — 별도 ADR

### 4. 결혼중개업법 적용 여부
- **외부 자문 후 결정** (BACKLOG P0 — 사용자가 변호사/관청 확인)
- "결혼" 직접 표방 안 함 → 일반 데이팅 앱 카테고리 유지 가능성 높음
- 보수적으로 **국내결혼중개업 신고 검토** 권장 — 신고 비용 작고 분쟁 시 안전판
- 결정 후 ADR 0008 (예정) 에 기록

### 5. 개인정보 / PII
- 탈퇴 시 즉시 삭제 (사진/프로필/연락처)
- 매칭/결제 이력은 분쟁 대비 90일 / 법정 5년 보관
- 사용자 노출 정책 (이용약관, 개인정보처리방침) 은 정식 출시 전 별도 페이지 게시 — 변호사 검수

### 6. 사용자 책임 / 한계
- 만남 후 결과 보장 없음
- 사용자 간 분쟁은 명백한 약관 위반만 운영자 개입
- 허위 정보 / 영업 목적 / 미성년자 → 즉시 차단
