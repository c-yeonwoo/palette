# 0029 — 무지개 카드 커버 복귀 + 컬러궁합 배너 제거 + 나의 색 위치

- **상태**: Accepted (ADR 0027 일부 대체)
- **결정일**: 2026-06-07
- **결정자**: ys.choi

## 컨텍스트

소개(홈) 추가 피드백:
- ADR 0027의 "상대 색 단색 워시" 커버가 원하던 게 아님 — 예전 **무지개(팔레트) 느낌**을 원함.
- "오늘의 컬러 궁합" 배너(`DailyMatchBanner`)의 점수가 잘리고 균형이 계속 어긋남 → 제거 요청.
- 마이페이지 내 프로필의 "나의 색" 섹션이 탭 안쪽에 묻혀 있음.

## 결정

1. **카드 커버 = 팔레트 무지개 워시.** 단색 워시 폐기. 8색 컬러타입 hue(빨/주/노/초/파/보/핑크)를 파스텔(α≈0.4) 대각 그라디언트로 섞어 흰 배경 위에 깔고, "탭해서 열기"·티저·궁합%는 프로스트 화이트 칩으로 올려 가독성 확보. `MainFeedScreen`의 모듈 상수 `PALETTE_COVER_STYLE`로 단일화(ProfileCard·AiSignalCard 앞면·잠긴 카드 공통).
   - 과거 `paint-overlay.png`는 리포에 커밋된 적이 없어 복원 불가 → CSS 무지개로 재구성(팔레트=색 전체 컨셉에 더 부합).
2. **컬러궁합 배너 제거.** `DailyMatchBanner`를 홈에서 미렌더(컴포넌트 파일은 향후 정제 재사용 위해 보존, import 제거).
3. **나의 색 위치 이동.** `MyProfileScreen`에서 "나의 색" 섹션을 탭 콘텐츠 밖, 열람수 stats 바로 아래(탭 위)로 이동 — 탭 무관 상시 노출.
4. **추천 탭 표기 "팔레트 Pick"** (한글 "픽"→영문 "Pick"), 백엔드 식별자(ai-signal·AiPassSubscription) 불변. (ADR 0028 연장)
5. **팔레트 Pick 카드 하단 캡션 제거** — 각 카드 아래 `rec.reason`("프로필 궁합도 기반 추천") 캡션 삭제(중복·노이즈).

## 결과

- 잠긴/미오픈 카드가 무지개 팔레트 커버로 통일, 텍스트는 프로스트 칩으로 가독.
- 홈에서 컬러궁합 배너 사라져 상단 정리.
- 나의 색이 프로필 상단(열람수 아래)에 상시 노출.
- vite build / vitest(40) 통과.

## 메모

- ADR 0027의 "색 단색 워시 커버"·"컬러궁합 배너 inset 정렬"은 본 ADR로 대체.
- `DailyMatchBanner.tsx`는 미사용 상태로 잔존(데일리 포춘/궁합 기능 재도입 시 정제 후 재사용 가능).
