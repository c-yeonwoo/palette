# ADR 0077 — 홈 ‘오늘’ 일일 질문 (리텐션 MVP)

- 상태: 채택
- 날짜: 2026-07-14
- 관련: [[0037]] (점진 인사이트), [[0041]] (리텐션 보너스)

## 배경

지인망이 얇은 유저는 피드가 비어 1~2회 방문 후 이탈하기 쉽다.
출석체크형 게이미피케이션은 Palette 톤과 안 맞으므로,
**“팔레트가 나를 알아가는 하루 한 질문”** 으로 매일 방문 이유를 만든다.

## 결정

- 홈(MainFeed) 헤더 직후 **TodaySection**
- 질문 풀은 코드 SoT (`DailyQuestionPool`, dayOfYear 회전)
- 답변은 `daily_answers` (user_id + answer_date UNIQUE)
- 완료 시 보너스 **물감 +2** (7일 만료), 하루 1회
- 스트릭: 연속 답변 일수 (오늘 미답이면 어제까지)
- 답변 후 CTA: 팔레트 Pick 탭으로 유도

## Non-goals (후속)

- 주간 미션 보드, 초대 densify 미션
- 답변을 프로필/매칭 임베딩에 반영
- LLM 일일 인사이트
