## 가격 정책 — 결정 사항

**현행 유지 (베타 단계)**. 정식 결제 활성화 시점에 재검토.

미래 방향 메모 (확정 X, ADR 0039 후속):
- **소개 요청 티켓 번들 가격 인상**: 1장 10,000원 / 5장 40,000원 / 10장 80,000원
  → 현재 3,000원 대비 3배 인상, 진지함·신중함 강조
- **AI 추천 (팔레트 Pick 패스) 가격 인상**: 9,900원/월 → 추후 인상
- **초반엔 보너스·체험권으로 할인 폭 ↑** — 결제 진입 friction 해소가 우선

## P0 — 정식 출시 (3개월) 전 필수

### 수익화

#### B-001. 신규 가입 7일 무료 체험 ⭐
- 가입 직후 자동 지급: 프로필 열람 티켓 3장 + 소개 요청 티켓 1장
- 만료 7일 (그 이후 잔여분 자동 소멸)
- 목표: 결제 진입 전 가치 학습 (Aha! moment)
- KPI: 가입 후 7일 내 매칭 시도율 / 결제 전환율

#### B-002. 친구 가입 보너스 ⭐
- 친구 1명 가입 시 양쪽에 열람권 1장 + 신뢰도 +5
- 어뷰징 방지: 동일 IP/디바이스에서 다수 가입 제한, 친구 수락 24h 내 신규 가입자만 카운트
- KPI: 가입자당 평균 친구 수, 1주차 retention

#### B-003. 주선자 마일스톤 보너스 ⭐
- 첫 매칭 +500P / 5건 누계 +2,000P / 10건 +5,000P / 20건 +10,000P / 50건 +25,000P
- 신규 친구 가입 시 +200P (B-002와 별개)
- 목표: Lv.1~2 주선자 동기 부여 (현재 매칭당 225~300P는 박함)
- KPI: 주선자 활성도 (월 매칭 시도 수), 다이아 도달율

#### B-004. 묶음 할인 결제 화면 노출
- 정책 §1.2b 의 1·5·10장 옵션을 결제 모달에 표시
- 5장 옵션에 "best value" 배지
- 베타엔 노출만 — 정식 결제 활성화 시 클릭 → IAP/Toss

### UX · Discover

#### B-005. 친구 초대 마법사 ⭐
- 가입 직후 5단계 온보딩 후 옵셔널 단계 추가: "지인 5명 초대로 시작"
- 카카오 친구 동기화 옵션 (옵셔널, 미동의 시 친구 코드만 노출)
- 스킵 가능, 단 "친구 가입 시 보상 받기" 안내

#### B-006. 데모 풀 (친구 0명일 때) ⭐ — 부분 구현 → P1
- ✅ 베타: `NoFriendsNudge` + 시드 유저 12명 (현재 동작)
- ⏸ 정식 출시: 옵트인 토글 ("신규 유저에게 내 프로필 익명 노출 OK")로 실 가입자 풀로 전환 → **B-114 (P1)**

#### B-007. 푸시 알림 트리거 5종 ⭐ — 일부 P0, 나머지 P1
- ✅ 친구 신규 가입 (`FRIEND_NEW_SIGNUP`) — 친구 수락 hook 즉시
- ✅ 매칭 후 D+1·D+3·D+7 (`POST_MATCH_NUDGE`) — MatchFollowUpScheduler 확장
- ⏸ 친구의 친구 풀 갱신 — 감지 비용 + 스팸 위험 평가 후 → **B-115**
- ⏸ 팔레트 Pick 일일 갱신 — 옵트인 정책 결정 후 → **B-116**
- ⏸ 주선 활동 주간 보고 — 집계 형식 결정 후 → **B-117**

## P1 — 1~3개월

### 수익화

- **B-101. 팔레트 Pick 패스 번들 재설계** — AI 추천 무제한 + 매월 열람 5장 + 궁합 리포트 + 우선 추천. 가격 인상 검토.
- **B-102. 가격 A/B 테스트** — 소개 요청 2,500/3,000/3,500 3-arm (정식 결제 활성화 시점)
- **B-103. IAP 통합** — Apple iOS 의무 (디지털 재화), Toss 는 웹·Android, 가격 통일 (iOS 마진 양보 또는 17% 인상)
- **B-104. 환불 어뷰징 가드** — 월 환불 5회 cap, 동일 주선자 거절 패턴 자동 플래그
- **B-105. 티켓 18개월 만료** — 게임머니 표준, 만료 30일 전 푸시 안내

### UX

- **B-110. 색 콘텐츠** — 주간 "이번 주 ○○색의 운세/궁합" + 색별 익명 모임/AMA (운영자 큐레이션)
- **B-111. 매칭 후 가이드** — 첫 메시지 추천 / 데이트 코스 (dateCode 활용) / 관계 단계 팁
- **B-112. 부정 경험 완충** — 거절 직후 "이건 당신 잘못이 아니에요" + 다른 추천 즉시 노출 (회복 동선)
- **B-113. 옵셔널 팁 동선 강화** — 매칭 성사 후 주선자 코멘트 표시 + 팁 버튼 (감정 동선)
- **B-114. 데모 풀 옵트인 전환** — 정식 출시 시점에 시드 유저 → 실 가입자 옵트인 풀로 교체. 사용자 설정에 "신규 유저 둘러보기 풀에 익명 노출" 토글.
- **B-115. 친친 풀 갱신 알림** — 친구의 친구가 5명 이상 신규 진입 시 푸시. 감지: 일일 배치 or Friendship.acceptedAt 기준. 스팸 cap (주 1회).
- **B-116. 팔레트 Pick 일일 갱신 푸시** — 매일 오전 9시. 옵트인 토글 (마이페이지 설정). 미접속 N일 이상 사용자만 발송 (활성 사용자엔 노이즈).
- **B-117. 주선자 주간 보고** — 일요일 저녁. 이번 주 받은 요청·승인·성사 건수 + 누적 포인트 변동. 주선자 활성 사용자만.
