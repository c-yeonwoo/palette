# Palette — Product Requirements (PRD)

> 제품 의도 / 핵심 가치 / 범위. **사람만 수정**.
> 워커가 "이 변경이 제품 의도와 맞는가" 확인할 때 참조.

---

## 1. 서비스 개요

**Palette** — 지인 네트워크 기반 데이팅 앱.

> "나의 색을 찾고, 너의 색과 조화를 이루다"
> "Find Your Color, Meet Your Match"

화가가 팔레트에서 자신의 그림에 어울리는 색을 고르듯, 각자의 개성·취향·이상형이라는 고유한 색을 찾아 조화로운 관계를 만든다.

### 1.1 문제 정의 (웨지)

팔레트는 **단면 데이팅앱이 아니라 양면 마켓플레이스**다:
- **공급 = 주선자(마당발).** 소개팅은 한국에서 가장 신뢰받는 만남 방식이지만 병목은 "능동적으로 주선해주는 친구"다 — 희소하고 빠르게 지친다.
- **수요 = 솔로.** 지인 소개의 안전망을 원하지만 활발한 주선자가 늘 곁에 있진 않다.

> **웨지: "또 하나의 신뢰형 데이팅앱"이 아니라 "마당발을 시스템으로 대체·증폭하고, 마당발이 없어도 그 신뢰 경험에 접근하게 하는 레이어."**
> 주선자의 피로·리소스를 줄이고(소개 요청 → 승인만 하면 되는 반응형 구조 + 주선자용 AI 추천), 솔로에겐 이상형과 가까운 새 인연 창구가 된다.

### 1.2 콜드스타트 전략 — 순차적 차별화 (ADR 0072)

지인 주선(신뢰 차별화)은 네트워크가 형성된 뒤에야 켜진다. 그 전까지는 **팔레트 Pick(AI 양방향 매칭)을 콜드스타트 가치 엔진**으로 삼는다 — 지인이 없어도 수도권·거리순·이상형 매칭된 실제 프로필을 day-1에 제공. 콜드스타트 사용이 친구 그래프 엣지를 만들어내게 설계(공통 지인 힌트·클러스터 초대)해, 그 위에 지인 주선 레이어가 활성화된다. **북극성 지표는 (초기) 네트워크 밀도 = 유저당 평균 2촌 풀 크기.**

## 2. 핵심 가치 (변하지 않는 것)

### 2.1 나만의 색깔 표현 (Express Your Color)
- AI 인터뷰로 프로필 자동 생성
- 동영상/사진 신뢰도 검증 시스템
- **8가지 색깔 메타포** 로 개성 표현 (Orange/Blue/Red/Pink/Green/Purple/Yellow/Gray)

### 2.2 조화로운 매칭 (Find Your Harmony)
- 1-2촌 지인 네트워크 기반 매칭
- **2단계 검증** (주선자 승인 → 수신자 승인)
- 건별 과금으로 신중한 선택 유도

### 2.3 의미있는 관계 (Create Masterpiece)
- 주선자 등급·명예 시스템 (레벨/리그·랭킹 — 무현금, ADR 0064)
- 매칭 후 **10일 쿨타임**으로 집중 유도
- 사회적 책임감 기반 관계 형성

## 3. 디자인 원칙

| 원칙 | 의미 |
|---|---|
| **무대 vs 작품** | brand 색 = 모노톤 무대, 8 컬러 = 사용자 작품 (ADR `0001`) |
| **진정성 > 화려함** | 둥글둥글 귀여운 톤보다 진중한 무게감 |
| **한글 우선** | 사용자 노출은 모두 한국어 ("팔레트", "소개받기" 등) |
| **신뢰의 표시** | 1촌 정보, 사진 검증, 매칭 단계 — 신뢰 신호를 UI 에 자연스럽게 |

## 4. 사용자 / 페르소나

### 4.1 일반 사용자 (REGULAR)
- 25-35세 직장인
- 지인 추천이라 (믿을 만한) 사람을 만나고 싶지만 부담 없게
- 본인 색깔(타입)을 통해 자기 표현하고 잘 맞는 사람을 찾고 싶음
- 신중한 만남을 선호 — 무한 스와이프 피로 ↓

### 4.2 주선자 (MATCHMAKER_ONLY 또는 REGULAR 가 겸업)
- 평소 친구들 사이에서 "사람 잘 연결한다" 평 듣는 사람
- 친구들의 좋은 인연을 만들어주는 보람 (금전 보상 없음 — 무현금 호의)
- 레벨이 오를수록 명예 ↑ (리그/랭킹)

## 5. 핵심 흐름 (사용자 여정)

### 5.1 일반 사용자
1. 베타 코드 입력 → 가입
2. 휴대폰 인증
3. 프로필 작성 4단계 (BasicInfo → Photo → AboutMe → IdealType)
4. AI 가 색깔 + 소개글 생성 (사용자 답변 인용한 근거 + 500자+ 소개)
5. 친구 초대 / 초대 코드 (1촌 확장)
6. 피드에서 2촌 프로필 탐색 (필터: 나이/키/지역/직업). **지인망이 얇으면(콜드스타트) 팔레트 Pick 이 수도권·거리순 공개 발견 풀로 실제 프로필을 채운다 — ADR 0072**
7. AI 시그널 (1+1) 매일 확인
8. 마음에 드는 사람 → 1촌 주선자 선택 → 매칭 요청
9. 주선자 승인 → 수신자 승인 → 연락처 교환
10. 매칭 후 10일 쿨타임

### 5.2 주선자
1. 일반 가입 또는 주선자 전용 가입
2. 주선자 대시보드 — pending 요청, 친구 멤버, 누지 (추천)
3. 요청 승인/거부 (이유 코멘트 가능)
4. 성공 시 주선자 등급 상승 (금전 보상 없음 — 무현금, ADR 0064)
5. 리그/랭킹에서 본인 위치 확인 (티어: Bronze/Silver/Gold/...)

## 6. Phase 별 범위

### Phase 1 (0-3개월) — MVP ✅ 베타 진행 중
- 회원가입 + 프로필 생성 + AI 분석
- 사진/영상 업로드 (S3)
- 1촌/2촌 네트워크
- 주선 요청 2단계 승인
- 주선자 대시보드 + 레벨
- 휴대폰 인증 (베타 bypass)
- 푸시 알림 이벤트 (mock)
- 리그/랭킹
- **콜드스타트 공개 발견 풀 + 시군구 거리 (진행 중 — ADR 0072)**: 지인망 얇을 때 팔레트 Pick 폴백. 기존 프론트 데모 시드 12명 하드코딩을 엔진 레벨로 대체

### Phase 2 (3-6개월) — 진행 예정
- AI Signal 결제 (Toss 검증)
- 채팅 (WebSocket/SSE)
- 알림 화면 백엔드 연결
- 사진 신원 검증 (Vision API)
- NICE 본인인증 실키
- NCP SMS 정식
- Apple/Google/Naver OAuth
- Capacitor iOS/Android 패키징
- App Store / Play Store 출시
- **내친소 게시판** — 지인 추천형 디스커버리 (주선자가 지인 추천 → 신청 → 선택 → 2단계 매칭). 신뢰 기반 "셀소" 변주. 유동성 확보 후 (ADR 0071)

### Phase 3 (6-12개월) — 장기
- **벡터 유사도 기반 AI 추천** (현재는 랜덤)
  - 프로필 소개글 + 이상형을 임베딩
  - pgvector / Pinecone
  - 코사인 유사도 top N
- 그룹 매칭
- **WANTED 역경매** — 이상형 게시 → 지원 → 선택 (내친소 머신 재사용, a/b 실험, ADR 0071)
- CloudWatch / CDN / Loki
- React Native 리라이트 검토 (Capacitor 한계 시)

## 7. 비즈니스 모델

### 7.1 결제 모델 — 단일 잔액 "물감" (ADR 0042 + 0044)

**모든 결제는 단일 잔액(물감)에서 차감.** `1 물감 = 100원`. 사용자 노출 단위 "물감" — 브랜드(팔레트) 정체성. 코드 변수 P (Paint) 유지.

| 항목 | 가격 | 베타 | 정식 | 메모 |
|---|---|---|---|---|
| 1촌 프로필 열람 | 무료 | ✅ | ✅ | 이미 아는 사이 |
| 친구의 친구 프로필 열람 | **20 물감** (2,000원) | ⚠️ 결제 미연동 | ✅ | 커피 한잔값 |
| 한 다리 더 건너 | **30 물감** (3,000원) | ❌ | Phase 2 | 셀프 풀 확장 인센티브 |
| **소개 요청** | **100 물감** (10,000원) | ⚠️ 결제 미연동 | ✅ | 매칭 시도 1회당 진심 신호 |
| **팔리의 연애 리포트** 열람 | **1 물감** (100원) | ✅ | ✅ | 1회 해제 후 영구 · 리워드 광고 단가와 등가(floor) · "광고 보고 무료" 준비 중 (ADR 0069/0070) |
| 팔레트 Pick 패스 | **29,900원/월** | ⚠️ paymentKey 만 체크 | ✅ Toss 검증 | 첫달 무료 (ADR 0045) |

> 주선자 금전 보상(커미션 분배·팁·출금)은 무현금 모델로 제거 — 주선은 등급·명예 기반 호의 (ADR 0064 / CM-003). 수익은 100% 사용자 측(열람·소개요청·구독).

### 7.2 보너스/체험 (ADR 0041 + 0042 + 0044 + 0048)

| 종류 | 수량 | 유효 | 트리거 |
|---|---|---|---|
| 가입 환영 보너스 | **60 물감** (친친 3명 열람 가치) | 7일 | 가입 직후 자동 |
| **초대 코드 가입 보너스 (양쪽)** | **100 물감** (친친 5명 열람 가치) | 30일 | 유효 초대 코드로 가입 시 양쪽 적립 + 자동 1촌 형성 (ADR 0048) |
| 친구 가입 보너스 (양쪽) | **10 물감** | 14일 | 친구 수락 시 (초대 코드 없이 친구 요청·수락) |
| **신규 트라이얼** (ADR 0045) | 3일 일 5명 무료 열람 + 110물감 묶음 반값 1회 + 무료 소개 1회 (7일 만료) + 팔레트픽 첫달 무료 | — | 가입 직후 자동 부여 |

### 7.3 충전 묶음 (현행 베타)

| 받는 물감 | 결제 금액 | 보너스 |
|---|---|---|
| 5 물감 | 500원 | — (최소 충전 — 리포트 등 소액용) |
| 50 물감 | 5,000원 | — (정가) |
| **110 물감** | 10,000원 | +10 (10%↑) |
| **340 물감** | 30,000원 | +40 (13%↑) BEST VALUE |
| **575 물감** | 50,000원 | +75 (15%↑) MAX |

> 충전 잔액(`paidPoints`)은 만료 없음. 보너스 잔액(`bonusPoints`)은 유효기간 후 만료. 소비 순서 bonus → paid.

### 7.4 주선자 보상 — 명예 + 폐쇄형 크레딧 (ADR 0064 + 0072 확정)

- 주선자는 플랫폼으로부터 **현금**을 받지 않는다. 보상은 **(a) 명예**(리그·랭킹·"내가 이어준 커플" 스토리) **+ (b) 폐쇄형 크레딧**(성사 시 물감/열람권/구독 크레딧 — 다시 플랫폼 내 데이팅에 소비).
- 현금 출금·90/10 팁·커미션은 채택하지 않음(컴플라이언스 회피 + 재사용 루프). 출금 도메인(`WithdrawalRequest`)은 휴면 보존(전환 대비) — 백엔드 정리는 CM-003.
- 크레딧 규모·트리거 상세는 후속 스프린트에서 확정(BACKLOG).

> 정식 결제 활성화 시점에 가격 재검토 (`BACKLOG` PA-sprint). 미래 방향은 IAP 활성화 후 ADR 별도 작성.

**상세 정책**: `docs/POLICY.md` · `docs/BACKLOG.md`
